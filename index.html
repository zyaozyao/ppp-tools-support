扩展使用

准备工作（初级 & 进阶都需） 准备工作教程

  • 一台可运行 Chrome 浏览器的电脑
  • 安装 Chrome 浏览器
  • 安装「PPP Tools」扩展
  • 安装「TokenPocket」扩展（经过综合比对，目前TokenPocket是最佳选择）

初级模式

  • 访问 ppp.fun 连接 TokenPocket 钱包
  • 进入 奖励页，点击右上角「Enter Code」输入推荐码 19YO8Y6T 并保存。
  • 随便进入一个项目详情页面，点【QUICK BUY】->点【Settings】-> 勾选【Random】、勾选【Faster Confirmation】->Save Config
  • 打开「PPP Tools」扩展面板，设置提前抢购秒数[推荐30～40，因为签名有时效性，太长可能会失效]、设置快速购买最大值[跟官网保持一致，目前10]、设置要监控的PPP项目CA、最大抢购张数（可多个项目），并保存。
  • 接下来扩展会自动监测项目进度、提前5分钟左右根据配置的最大抢购数量打开多个项目页、自动滚动加载页面、切换页面、实时监测倒计时、倒计时结束前x秒(扩展配置的)自动进行点击quick buy、自动设置最大抢购数量、自动点击confirm按钮
  • 自动点击快速购买弹窗中的confirm按钮后，会弹出钱包确认弹窗，这个需要手动来点，初级模式这里是无法自动化点钱包弹窗里的确认的

初级模式说明：

  • 扩展在自动打开项目页面后会检测钱包与ppp.fun的连接状态并尝试自动连接、检测推荐码绑定情况并提示。
  • 进入一个项目详情页面，点【QUICK BUY】->点【Settings】-> 勾选【Random】、勾选【Faster Confirmation】->Save Config 这个操作只需要做一次就会缓存在本地，不用每次都这样做，除非你清空浏览器缓存数据
  • 手机上有一个【狐猴浏览器】，它可以安装chrome浏览器扩展，用它访问chrome商店搜索"PPP Tools"就可以找到并安装，对于没有电脑的用户可以通过这种方式来使用，总会比你手动快，不过还是建议使用电脑，电脑总比手机强大的

初级模式操作简单、容易上手，可熟悉流程；进阶模式则完全自动化，让你抢图如飞！

进阶模式 进阶模式指导教程

  • 电脑安装 Node.js 及puppeteer
  • 下载并修改自动化点击脚本配置
  • 运行脚本调起浏览器, 解锁钱包（第一次运行要创建或这导入钱包）
  • 在脚本调起的浏览器中访问 ppp.fun 连接 TokenPocket 钱包
  • 在脚本调起的浏览器中访问 奖励页，点击右上角「Enter Code」输入推荐码 19YO8Y6T 并保存。
  • 在脚本调起的浏览器中随便进入一个项目详情页面，点【QUICK BUY】->点【Settings】-> 勾选【Random】、勾选【Faster Confirmation】->Save Config
  • 在脚本调起的浏览器中打开「PPP Tools」扩展面板，设置提前抢购秒数[推荐30～40，因为签名有时效性，太长可能会失效]、设置快速购买最大值[跟官网保持一致，目前10]、设置要监控的PPP项目CA、最大抢购张数（可多个项目），并保存。
  • 接下来扩展会自动监测项目进度、提前5分钟左右根据配置的最大抢购数量打开多个项目页、自动滚动加载页面、切换页面、实时监测倒计时、倒计时结束前x秒(扩展配置的)自动进行点击quick buy、自动设置最大抢购数量、自动点击confirm按钮
  • 进阶脚本会自动快速点击钱包弹窗中的确认按钮，完成签名，等倒计时结束后自动发送交易，抢购完成。
  • 前面扩展自动打开的项目页面会在5分钟后自动关闭，然后等待下一轮自动抢购

进阶模式说明：

  • 进阶模式与初级模式相比主要多了一个自动化脚本的运行，这个脚本的用处就是自动快速点击初级模式中最后一步的钱包签名确认弹窗的确认按钮从而彻底解脱双手。其他流程均与初级模式一致
  • 进阶模式中会自动监控钱包弹窗并自动点击确认按钮，所以在脚本调起的浏览器中，除了解锁钱包、连接钱包到PPP官网、及设置钱包锁定时间外尽量不要手动做其他操作，以防误操作。
  • 进阶模式的脚本还会定时打开钱包页面（10分钟一次）然后再自动关闭，以此来防止钱包休眠锁定，这样就可以放心托管，彻底解放人力
